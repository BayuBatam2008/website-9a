# Memulai
### 1. Copy Git Ini Ke Lokal

```
git clone https://github.com/BayuBatam2008/website-9a/
```

### 2. Masuk Ke Dalam Direktori

```
cd website-9a
```

### 3. Install Paket Yang Diperlukan

```
npm install
```

### 4. Edit _data/metadata.json

### 5. Jalankan Eleventy

```
npm start
```

Debug Mode:
```
DEBUG=* npx eleventy
```

Build:
```
npm run build
```
