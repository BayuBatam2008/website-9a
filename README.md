# Getting Started

### 1. Navigate to the directory

```
cd website-9a
```

### 3. Install dependencies

```
npm install
```

### 4. Edit _data/metadata.json

### 5. Run Eleventy

```
npm start
```

In debug mode:
```
DEBUG=* npx eleventy
```

To build 
```
npm run build
```
